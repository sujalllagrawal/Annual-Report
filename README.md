#  Annual Report Analysis (AI + Excel Automation)

This project automates the fundamental analysis of any company's annual report using AI and Microsoft Excel.

  - Balance Sheet

- The cleaned data is exported and uploaded into **Microsoft Excel**.
- Excel automatically calculates:
  - Important ratios (P/E, P/B, ROE, ROCE, D/E, margins)

  - Competitor benchmarking
